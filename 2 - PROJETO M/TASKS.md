## A Fazer

- [ ] Definir a cosmologia básica do mundo.
- [ ] Esboçar a história de uma nação ou império principal.
- [ ] Criar 5 NPCs com motivações claras.
- [ ] Detalhar uma cidade principal: geografia, cultura, pontos de interesse.
- [ ] Desenvolver um sistema de magia inicial com suas regras e limites.
- [ ] Adicionar 3 novas criaturas ao bestiário.
- [ ] Explorar potenciais conflitos entre facções políticas.
- [ ] Explorar lendas e mitos populares em diferentes culturas.
- [ ] Explorar o impacto de uma tecnologia ou magia rara na sociedade.
- [ ] Criar entradas para Continentes/Regiões (ex: Nome do Continente 1, Nome da Região 2).
- [ ] Criar entradas para Cidades & Pontos de Interesse (ex: Cidade Principal A, Ruína Antiga B, Masmorra do Dragão C).
- [ ] Preparar/Linkar mapas (ex: Mapa Geral do Mundo, Mapa do Continente X).
- [ ] Criar entradas para Cultura (ex: Festival da Colheita, Culinária Élfica).
- [ ] Criar entradas para Religiões (ex: Panteão dos Antigos, Culto da Sombra).
- [ ] Criar entradas para Política (ex: Reino de Eldoria, Guilda dos Mercadores).
- [ ] Criar entradas para Economia (ex: Rota Comercial da Seda, Minas de Mithril).
- [ ] Criar entradas para NPCs Importantes (ex: Rei Theron, Arcimaga Elara, Ladrão Sombra Noturna).
- [ ] Criar entradas para Grupos & Etnias (ex: Elfos da Floresta, Anões das Montanhas, Nômades do Deserto).
- [ ] Criar entradas para Genealogia (ex: Casa Real de Valerius, Linhagem dos Primeiros Reis).
- [ ] Criar entradas para Sistemas de Magia (ex: Magia Elemental, Conjuradores de Sangue).
- [ ] Criar entradas para Tecnologia (ex: Motores a Vapor, Armas de Pólvora).
- [ ] Criar entradas para Bestiário (ex: Grifo Real, Dragão das Sombras, Flor da Lua Crescente).
- [ ] Criar entradas para Cronologia (ex: Era dos Primórdios, Época Dourada, Idade das Trevas).
- [ ] Criar entradas para Grandes Eventos (ex: A Grande Guerra dos Cem Anos, A Queda de Veridia, Descoberta da Magia Arcana).
- [ ] Criar entradas para Lendas (ex: O Herói Esquecido, A Profecia do Fim, O Segredo da Floresta Encantada).

### 2. Regras de Escrita
- **Consistência:** Mantenha a grafia de nomes de locais, personagens, facções e termos específicos sempre iguais.
- **Formatação de Títulos:** Use `#` para títulos principais, `##` para seções, `###` para subseções.
- **Links Internos:** Utilize links `[[Nome do Arquivo]]` ou `[Texto do Link](Caminho/para/arquivo.md)` para referenciar outras notas.
- **Listas:** Use `*` ou `-` para listas não ordenadas e `1.` para listas ordenadas.
- **Destaque:** Use `**negrito**` para termos importantes e `*itálico*` para ênfase ou nomes de obras.
